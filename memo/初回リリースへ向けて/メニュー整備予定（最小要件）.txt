メニュー整備予定（最小要件）

#main-menu
pre-generationへ ボタン

#pre-generation
（シード値自動設定）
シード値表示
世界の生成 ボタン
→地形生成・スポーン

#ingame-menu
- Menu ボタン
  → ルートメニュー開閉

- ルート
  - ObjectList
  - Generation

- ObjectList
  → オブジェクト一覧を表示
  → grass / herbivore / predator のカテゴリ切替
  → 一覧から対象選択
  → 選択対象の追跡
  → 追跡中は対象追跡カメラとして動作
  → 動物上ゲージ表示を観測に利用
  → 状態表示 UI を併用
  → detail ボタンで詳細表示

- Generation
  → 世代更新系メニューを表示
  - AdvanceGeneration
    → 一斉世代更新 UI を表示
    → 初回リリースでは実行トリガー中心
  - GenomeViewer
    → DNA ビューアを表示
  - GenomeInjector
    → DNA 文字列入力からスポーン
    → Herbivore / Predator 両対応

- ingame-menu 外の常時機能
  → 動物上ゲージ表示
