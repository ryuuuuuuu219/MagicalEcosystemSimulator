初回リリース機能候補一覧

最小要件前提で、初回リリースでは「今すでに動いている核」を公開対象にする。
細かい公開判断や拡張前提の機能は中期タスク側へ回す。

## 初回リリースに含める機能

- ランダム地形生成
- 草地生成
- 草食動物の探索と摂食
- 肉食動物の探索、追跡、攻撃、摂食
- 死亡と分解
- 炭素循環
- オブジェクト一覧
- オブジェクト選択
- 対象追跡カメラ
- 状態表示 UI
- DNA ビューア
- 一斉世代更新
- 草食動物 / 肉食動物 Genome Injector
- 動物上ゲージ表示

## 初回リリースから外す機能

- `threat map`
- `heat field`
- カメラ追従方式の切替 UI
- predator DNA export
- 一斉世代更新の詳細設定 UI
- 動物上ゲージの表示切替 UI
- 高度な環境調整 UI
- 実験条件の保存 / 再現用 UI

## 完了条件

- 起動から観察開始まで進められる
- オブジェクト観察、追跡、状態確認ができる
- 一斉世代更新が実行できる
- Genome Injector が使える

## 補足

- `パフォーマンス監視` は存在するが、初回リリースの主機能ではなく補助扱い
- 詳細設定付きの世代更新や debug 可視化は、初回リリースでは扱わない
