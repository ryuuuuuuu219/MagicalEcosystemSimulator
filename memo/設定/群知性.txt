◇航空作戦で使われる 目標識別レベル（target identification level）
enum TargetStatus
{
    Unknown,   // 未観測
    Bogey,     // 未確認目標
    Suspect,   // 敵性疑い
    Hostile,   // 敵
    Friendly   // 味方
}

◇IFF（Identification Friend or Foe） 
「speciesID」種族ID
「factionID」派閥ID

if (target.speciesID == self.speciesID)
    status = Friendly;

ただし派閥がある場合は

if (speciesID same && factionID different)
    status = Suspect;

◇敵確定の条件
Hostile判定は次のどれかで起こします。

・攻撃された
・味方が攻撃された
・種族敵対
・距離識別
・以前の恨み記録あり
・指揮者が敵指定した

◇やってみたいこと
執念深さが忠誠度を上回り命令不服従を起こす
指揮者が過去に恨みを蓄積させた個体に向かって一斉攻撃を命ずる展開


◇基本ルール
speciesID same ならIFF上は原則 Friendly
ただし factionID different なら Suspect または Rival
commandPower が近い個体同士は派閥候補
loyalty が低い個体は追従率低下
rebelliousness が高い個体は命令を部分拒否

◇派閥形成条件の例
指揮者との commandPower 差が小さい
loyalty が低い
目標評価が食い違う
味方被害が増えている
指揮者が自分を盾に使った
