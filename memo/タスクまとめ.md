# タスクまとめ

このファイルは、`memo/タスク一覧/` 配下の作業を俯瞰するための入口メモ。
詳細仕様や実装対象は、各タスクフォルダの `README.md` と `設定：*.txt` を参照する。

## 優先タスク系

1. `0.遺伝子設計変更`
- 観測しやすい UI 構成への整理。
- 説明書整備、UI 着手順位の固定、Ingame UI 基盤の実装が主対象。

2. `1.生態系コア強化`
- 行動ベクトルと環境場を強化し、生態系の見え方を改善する。
- threat map / heat field / 地形差を行動判断へ統合する段階。

3. `2.挙動AI分離`
- AI の責務を分離し、挙動拡張しやすい構成へ寄せるための作業。

4. `3.戦闘システム拡張`
- 戦闘まわりの仕様整理と実装拡張を扱う。

5. `4.魔素基盤構築`
- 魔素や資源循環の基盤を整備するためのタスク。

6. `5.属性魔法拡張`
- 属性ごとの魔法挙動や関連仕様を拡張するためのタスク。

7. `6.群知性導入`
- 個体単位ではなく群れ単位の振る舞いを導入するためのタスク。

8. `7.学習と進化`
- 学習・進化要素をゲーム進行へ接続するためのタスク。

9. `8.支配種到達条件`
- 支配種の成立条件や到達判定を整備するためのタスク。

10. `9.課題ステージ`
- 通常課題ステージの仕様・実装管理。
- クリア条件、敵対勢力、報酬、観測 UI が主対象。

11. `10.課題ステージ（期間イベント_環境外乱つき）`
- 期間イベントや環境外乱を含む拡張課題ステージの管理。

- `0.遺伝子設計変更` は、現行ゲノム構造とDNA形式を整理し、他タスクの遺伝子拡張の入口も兼ねる。
- `1.生態系コア強化` は、生物挙動の見え方に直結する中核タスクとして位置づけやすい。
- `9` 系は課題ステージ運用、`10` はその拡張イベント系として読むと把握しやすい。
