# ROADMAP

この文書は `memo/` 配下の各メモを整理し、実装順と設計課題を一本化したロードマップです。
実装済みの基準は [README.md](/c:/魔法環境シミュ/README.md) を優先し、詳細な論点は各メモを参照します。

## 方針

1. 先に設計の共通前提をそろえる。
2. 次に生態系コアを強化し、観察可能な差を増やす。
3. その上で戦闘、群知性、魔法、支配種へ広げる。
4. UI は独立機能ではなく、観測と調整の基盤として並行整備する。

## 現状の整理

### 実装済み、または土台があるもの

- 草食・肉食の基本行動ループ
- threat map の計算と debug 可視化
- heat field の計算と debug 可視化
- 遺伝子閲覧、DNA 入力、世代更新 UI
- オブジェクト一覧、注目、状態表示 UI
- 動物上ゲージの描画基盤

### 未実装、または設計止まりのもの

- mana field / 魔素場
- 群れ意思決定と `GroupIntent`
- 種族 ID / 派閥 ID / IFF
- biome の体系化
- 属性魔法
- 支配種到達条件
- 統合された環境調整 UI

## リリース後更新

## リリース後フェーズ 1: 生態系コアの強化

### 目的

- wander の見え方を改善し、地形や環境差に応じた行動の意味を増やす。

### 主な対象

- 縄張り / 拠点ベース巡回
- 座標の好感度
- 高所、見晴らし評価
- 森の追加
- 視界隠蔽
- 肉食の待ち伏せ
- 追跡放棄条件

### 現状

- 餌、脅威、境界、wander の加重合成は実装済み。
- 地形法線補正、evasion、zigzag、追跡停止閾値なども一部実装済み。
- threat map は部品実装済みだが、主移動ロジックへの全面統合は未完。
- heat field を安全行動へ使うロジックは未実装。

### 参照

- [動物の移動・場（map）のまとめ.txt](/c:/魔法環境シミュ/memo/動物の移動・場（map）のまとめ.txt)
