# ROADMAP

この文書は、今後の実装計画だけを扱うロードマップです。  
実装済みの要素一覧は [README.md](/c:/魔法環境シミュ/README.md) を基準にします。

## 方針

今後は、既存の生態系ループを安定化した上で、

1. 行動の観察性を上げる
2. 集団性と戦闘性を強化する
3. 魔素と魔法の基盤を作る
4. 支配種進化へ接続する

という順で拡張する。

## Phase 1: 現状安定化

優先項目:

- 仮想ゲージ表示
- AI の計算量改善
- 世代交代 UI の保存、取出し、入力導線の整理
- 目標性能の再確認

基準:

- `60fps`
- 草 `1000`
- 草食 `100`
- 捕食 `30`

(追記：
フェーズ5時点で捕食者100
フェーズ9で上位捕食者15
フェーズ10で支配種最大3)

主な課題:

- 全走査の削減
- 記憶探索の線形コスト削減
- UI 上で状態を追いやすくする

## Phase 2: 生態系コアの強化

目的:

- wander を観察しやすい行動へ変える
- 草原と森林で意味のある行動差を出す

追加候補:

- 縄張り / 拠点ベース巡回
- 座標の好感度
- 高所、見晴らし評価
- 森の追加
- 視界隠蔽システム
- 肉食の待ち伏せ
- 追跡放棄条件

## Phase 3: 死体フェーズと菌類

目的:

- 分解ループを観察可能にする
- 分解者を追加して生態系を閉じる

追加候補:

- `Alive / Dying / Corpse / Decomposed` の分離
- 死体表示の明示化
- 菌類の追加
- 菌類による炭素返却補助

## Phase 4: 戦闘システム拡張

目的:

- 肉食戦闘を単純追跡から段階的判断へ拡張する

追加候補:

- 肉食動物 対 肉食動物戦闘
- 種族分け
- 高脅威判定
- 撤退条件
- 包囲
- 追い込み
- 囮

導入候補データ:

- `speciesID`
- `factionID`
- 敵性判定
- IFF 的な識別

## Phase 5: 群知性

目的:

- 単体 AI から集団 AI へ拡張する

追加候補:

- 指揮者
- 指揮力
- 忠誠度
- 反抗性
- 派閥形成
- 集団意図共有

想定行動:

- 攻撃
- 包囲
- 追い込み
- 囮
- 退却
- 散開
- 指揮者防護

## Phase 6: 魔素基盤

目的:

- 魔法より先に、魔素を環境場として成立させる

追加候補:

- 魔素グリッド
- 拡散
- 吸収
- 消費
- 濃度差による能力補正

補足:

- `経済.txt` の energy 接続案は後段
- 風属性はベクトル場として分ける余地がある

## Phase 7: 属性魔法

目的:

- 見た目と効果が結びついた属性行動を導入する

初期候補:

- 火
  - 継続ダメージ
  - 延焼
- 氷
  - 棘生成
  - 鈍足
- 雷
  - 瞬間攻撃
  - 連鎖
- 風
  - ノックバック
  - 拡散補正
- 空間
  - スタック脱出
  - 位置補正

## Phase 8: 魔法学習と進化

目的:

- 魔法を使うだけでなく、習得と進化のループを作る

追加候補:

- 使用確率
- 優先度
- 解放状態
- スキルツリー
- 覚醒
- 恐怖
- 飽き
- 自信
- 行動インデックス

進化対象:

- 魔法優先度
- 戦闘行動
- 脅威重み
- 属性耐性
- 代謝効率

## Phase 9: 支配種

目的:

- 支配種を「到達段階」として成立させる

追加候補:

- 支配種への進化
- 支配種の行動系
- 空間魔法を含めた戦闘アルゴリズム
- 高度行動能力の観測

到達条件候補:

- 魔素量
- 生存時間
- 戦闘勝率

## 直近の優先順

4. 森と視界隠蔽の実装
