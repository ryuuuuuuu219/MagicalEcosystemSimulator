# ROADMAP

この文書は、今後の実装計画だけを扱うロードマップです。  
実装済みの要素一覧は [README.md](/c:/魔法環境シミュ/README.md) を基準にします。

## 方針

今後は、既存の生態系ループを安定化した上で、

0. 設計不足を先に潰す
1. 行動の観察性を上げる
2. 集団性と戦闘性を強化する
3. 魔素と魔法の基盤を作る
4. 支配種進化へ接続する

という順で拡張する。

### 進捗確認 (2026-03-14):

- 部分定義済み
  - 時間スケール
    - `Update` と `FixedUpdate` の分離は実装されている
    - 世代単位は `generationIndex` と世代更新処理として存在する
    - ただし `frame / step / generation` の文書化は未了
  - 空間スケール
    - `WorldGenerator` に `terrainSize` と `heightmapResolution` がある
    - `threatmap_calc` に `scoreCellSize` と `lowThreatProbeDistance` がある
    - `HeatFieldManager` に `gridResolution` がある
    - ただしワールド全体の共通単位としては未統一
  - AI 更新モデル
    - 草食、肉食とも `Update` で認知とベクトル計算を行い、`FixedUpdate` で移動と消費を処理している
    - threat score は `threatmap_calc.Update()` で減衰している
    - ただし「毎フレーム、定周期、イベント」の全体方針は未文書化
  - 行動決定アルゴリズム
    - 草食、肉食とも `ComputeTotalVector()` で食物、脅威、境界、wander の加重合成を実装済み
    - ただし計算式の仕様書化と共通表現は未了
  - 進化評価式
    - `AdvanceGenerationController.EvaluateScore()` で `Random / Carbon / Health` を実装済み
    - ただし生存時間、摂食量、戦闘結果を含む fitness 拡張は未着手
  - 感覚モデル
    - 視覚、記憶、脅威距離、vision wave、wander wave は実装されている
    - ただし匂い、音、感覚統合パラメータは未定義
  - バイオーム
    - 草地と水位、地形傾斜は行動と生成に影響している
    - ただし草原、森などの明示的 biome 定義は未実装
  - 魔法エネルギー接続
    - `carbon -> energy` 変換と `heat` 付与は実装されている
    - `HeatFieldManager` に熱グリッドの拡散、減衰、可視化がある
    - ただし mana との接続は未着手
  - プレイヤー目的
    - 世代交代 UI、DNA ビューア、DNA 注入の観察導線はある
    - ただし最終目標はメモ段階で、README / ROADMAP 基準には未統合
  - デバッグ可視化
    - `ThreatMapsGenerator` に threat map の可視化がある
    - `WorldUIManager.StateView` に state weights と wave の可視化がある
    - `VirtualGaugeManager` に health / energy / carbon の表示切替がある
    - `PerformanceBudgetMonitor` に fps と負荷監視がある

- 未着手または未定義
  - 種族テンプレート
    - `speciesID` と `factionID` はメモ段階で、スクリプト実装は未確認
  - 魔素拡散式
    - mana / magic 用グリッドや吸収、減衰は未実装
  - 群れ意思決定
    - `GroupIntent`、指揮系統、派閥形成はメモのみで未実装

- 方針変更反映
  - 死体の腐敗進行や菌類追加は実装予定から撤廃済み
  - そのため Phase 0 では死体の数値設計より、既存の分解、炭素返却、熱放出の扱いを優先して整理する

## Phase 5: 魔素基盤

目的:

- 魔法より先に、魔素を環境場として成立させる

追加候補:

- 魔素グリッド
- 拡散
- 吸収
- 消費
- 濃度差による能力補正

概要:
魔素
スカラー場
行動コスト
空間魔法使用時に環境での総量が増加
これは全域の高さが一様に増える
生残中、かつ魔法への素養があれば周囲から吸収する

スカラー場は谷のように減る

魔法使用により「消失」し、自然に増加しない。

ところで環境が死骸を分解、消去するのは空間魔法の一種である
