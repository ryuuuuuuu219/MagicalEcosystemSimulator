リーダーから群れの構成個体にニューラルネットワークって感じのパーティクルシステムをつなぐ？
ニューラルネットワークって感じとは？二座標間に3*n(n>2)以上の点を置く。この際リーダーから順に３点ずつグループ分けし、この３点は三次元座標が重ならないよう球状にばらつかせる。この点たちをA/B/Cとグループ分けしたとき、nグループのA点はn-1グループのA点と同グループのB・C点とn+1グループのA点と相互にラインレンダラーで接続

同グループの３点を頂点とした二次元メッシュを混ぜたい気もする
