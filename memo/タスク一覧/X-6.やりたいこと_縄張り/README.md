# X-6.やりたいこと_縄張り

## 目的

- 縄張りを生態系コアの基礎実装から切り離し、将来的に独立コンポーネントとして拡張できる形にする。

## 将来拡張メモ

- 機能別コンポーネントとして独立したい。
- 縄張り意識は一定以上の知能をトリガーとして付与したい。
- 同じ相の個体との二等分線で区切った領域を定義したい。
- 領域は複数の頂点と、それを頂点とする三角形集合として規定・記憶したい。
- 縄張りに入った個体に対し、複数状態を保持する enum を持たせたい。

## 基礎実装との関係

- 1 段階目の円状縄張りと侵入察知は `2.生態系コア強化` 側で扱う。
- 本ファイルは、その先の高知能向け縄張り拡張を保管する。
