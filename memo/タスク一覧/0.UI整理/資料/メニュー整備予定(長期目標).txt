# メニュー整備予定(長期目標)

## 初回リリースから退避した項目

- カメラ追従方式の切替 UI
- predator DNA export
- 状態表示 UI 内のページ切り替え導線の整理
- 状態表示 UI からの DNA ビューア導線の整理
- 一斉世代更新の詳細設定 UI
- パフォーマンス監視のメニュー導線整理

最小要件の初回リリースでは、起動から観察開始、対象観察、世代更新、Genome Injector までを優先し、上記は中期タスク側へ退避する。

## レイアウト

- 予定:
  左右 1/6 の領域に収める
- 現状:
  `Canvas` 左上に固定配置のボタン群で構成。
  `Menu`、`list tab`、`gen tab`、各サブボタンは個別 `RectTransform` で配置されており、統一レイアウトコンテナ化は未実施。
  表示制御は `WorldUIManager` のメニュー木で管理しており、親ボタン押下で配下 UI の開閉は可能。
  トグル挙動と多重クリック抑止は実装済み。
- 関連:
  `Assets/script/UI/WorldUIManager.cs`
  `Assets/script/UI/Menu/WorldUIManager.MenuTree.cs`

# UI

## マウス選択のレイヤー

- 予定:
  選択対象レイヤーを切り替えられるようにする
- 現状:
  `WorldUIManager` に `selectableLayer` があり、クリック選択時の `Physics.RaycastAll` に使用中。
  ただし Inspector 設定のみで、メニューからの変更 UI は未実装。
- 関連:
  `Assets/script/UI/WorldUIManager.cs`

## 動物上ゲージ

- 予定:
  表示内容と対象を UI から制御できるようにする
- 現状:
  ワールド空間 `VirtualGaugeCanvas` を動的生成して表示中。
  選択対象や可視状態の更新処理あり。
  内部的には表示項目 ON/OFF、相ごとの表示 ON/OFF、透明度の概念がすでに存在する。
- 関連:
  `Assets/script/UI/Menu/WorldUIManager.VirtualGauge.cs`
  `Assets/script/UI/VirtualGaugeManager.cs`

### 表示ステータス選択（HP・炭素・エネルギー）

- 予定:
  表示する情報を切り替えられるようにする
- 現状:
  ゲージ表示自体はあるが、メニューからの表示項目切り替え UI は未実装。
  内部には `ShowHealthGauge`、`ShowEnergyGauge`、`ShowCarbonText` が存在する。

### 表示対象の相（草食動物・肉食動物）

- 予定:
  草食動物と肉食動物の表示対象を切り替えられるようにする
- 現状:
  内部的に草食・肉食を識別しているが、表示対象相の絞り込み UI は未実装。
  内部には `ShowHerbivoreGauges`、`ShowPredatorGauges` が存在する。

### 透明度

- 予定:
  ゲージ透明度を調整できるようにする
- 現状:
  透明度調整 UI は未実装。
  内部には `GaugeAlpha` が存在する。

# カメラ操作

## リストから注目

- 予定:
  一覧経由で対象に注目する
- 現状:
  実装済み。
  オブジェクト一覧から対象ボタンを押すと `SetTarget` で注目対象に設定される。
- 関連:
  `Assets/script/UI/Menu/WorldUIManager.ObjectList.cs`

### リスト展開

- 予定:
  リストを展開して対象を選べるようにする
- 現状:
  実装済み。
  `list tab` で一覧表示をトグル可能。
  一覧は右下寄りに動的生成され、カテゴリ切替あり。

#### ステータス閲覧

- 予定:
  リストから選んだ対象の詳細を見られるようにする
- 現状:
  実装済み。
  対象選択後に `detail` ボタンで状態表示 UI を開閉可能。
  草食・肉食で表示内容が分岐し、DNA コード表示や波形表示あり。
- 関連:
  `Assets/script/UI/Menu/WorldUIManager.StateView.cs`

#### ページ切り替え

- 予定:
  詳細表示ページを切り替えられるようにする
- 現状:
  実装済み。
  `<` と `>` で状態表示ページ切替可能。

### 注目時カメラ位置の設定

- 予定:
  注目時のカメラ配置パターンを選べるようにする
- 現状:
  部分実装。
  `RotationThenlooking` フラグで追従方式を切替。
  真上固定と、現在回転を維持した追従の 2 系統あり。
- 関連:
  `Assets/script/UI/WorldUIManager.cs`
  `Assets/script/UI/Menu/WorldUIManager.ObjectList.cs`

#### 真上

- 予定:
  真上視点を選べるようにする
- 現状:
  実装済み。
  `RotationThenlooking == false` で真上視点になる。

#### 現在値維持
未実装。
