# 予定

## Properties

- 数値入力やスライダーを使える設定行を追加する。
- Environment / UI / Camera / Generation / Deferred の表示名を日本語寄りにする。
- Performance 設定の reflection 依存を専用 API に置き換える。

## State View

- 選択中個体の現在目標を表示する。
- 追跡状態、攻撃可否、最後の攻撃時刻を表示する。
- 草食 / 捕食で共通して見る基本値のページ構成をそろえる。

## Camera

- 追跡カメラ ON/OFF を追加する。
- 指定個体と目標個体の構図プリセットを追加する。
- カメラ距離、FOV 係数、上方向オフセットを調整できるようにする。
- 詳細設計は `../カメラ/追跡カメラ設計.md` に置く。

## Generation

- 突然変異率、親由来変異範囲を直接入力またはスライダーで調整する。
- 草食 / 捕食の入力モードをプルダウンで選ぶ。
- 世代更新前に設定差分を確認する。

## 保留

- 戦闘ログ表示は `../../X-2.やりたいこと_判断ログ/README.md` に置く。
- 戦闘可視化用の実装タスクは `../README.md` に置く。
