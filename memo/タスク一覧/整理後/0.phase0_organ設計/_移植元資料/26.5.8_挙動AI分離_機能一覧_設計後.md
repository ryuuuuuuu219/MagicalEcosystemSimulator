# 26.5.8 挙動AI分離 機能一覧 設計後

作成日: 2026-05-08

## 目的

挙動AI分離後に存在させる機能を、役割ごとに一覧化する。

詳細な移行方針は `26.5.8_挙動AI分離_現状機能と設計方針.md` を参照する。

ここでいう organ は、生体器官を再現する `.cs` コンポーネントである。個体は `AddComponent` によって organ を獲得し、その獲得、欠落、level 変化を進化の演算対象として扱う。

## 全体構造

```text
AnimalBrain
  -> Context
  -> Sense
  -> Memory
  -> Desire
  -> Action
  -> Motor
  -> Gene / Installer
```

## Core

| 機能 | 役割 |
| --- | --- |
| `AnimalBrain` | Sense、Desire、Action、Motor を束ねる中核。 |
| `AIContext` | 個体、地形、Resource、mana、health、速度、manager 参照を共有する。 |
| `AIMoveIntent` | Desire が返す移動方向、重み、理由。 |
| `AIComponentGene` | 機能の enabled / level / tuning を遺伝対象として持つ。 |
| `AIComponentSet` | 個体に搭載するAI機能の集合。 |
| `AnimalAIInstaller` | manager から受け取った component set をもとに AddComponent する。 |

## Sense

外界を観測して、Memory または Context に情報を渡す。

| 機能 | 対象 | 役割 |
| --- | --- | --- |
| `FoodVisionSense` | 草食 | 視界内の草を検知する。 |
| `PredatorVisionSense` | 草食 | 視界内または検知範囲内の捕食者を検知する。 |
| `PreyVisionSense` | 捕食 | 獲物候補を検知する。 |
| `ThreatVisionSense` | 捕食/共通 | 上位捕食者などの脅威を検知する。 |
| `ThreatMapSense` | 共通 | 脅威度マップを読み、危険方向や危険度を得る。 |
| `ManaFieldSense` | 共通 | mana field の濃度と勾配を読む。 |
| `SafeFoodPathEvaluator` | 草食 | 草へ向かう経路が捕食者の危険圏に入るか判定する。 |
| `CreatureRelationResolver` | 共通 | speciesID、category、IFF から敵味方や捕食対象を判定する。 |

## Memory

Sense が得た情報を短期記憶として保持する。

| 機能 | 対象 | 役割 |
| --- | --- | --- |
| `AIMemoryStore` | 共通 | 対象と残り記憶時間を保持する。 |
| `FoodMemory` | 草食 | 餌の座標または GameObject を記憶する。 |
| `ThreatMemory` | 共通 | 脅威対象を記憶する。 |
| `PreyMemory` | 捕食 | 獲物対象を記憶する。 |
| `TargetTracker` | 捕食/共通 | 追跡対象の前回位置、現在速度、見失い状態を持つ。 |

## Desire

移動したい方向と重みを返す。最終的な合成は `AnimalBrain` が行う。

| 機能 | 対象 | 役割 |
| --- | --- | --- |
| `FoodDesire` | 草食 | 記憶または視界内の草へ向かう。 |
| `PreyChaseDesire` | 捕食 | 獲物へ向かう。 |
| `ThreatAvoidanceDesire` | 共通 | 脅威から離れる。 |
| `BoundaryAvoidanceDesire` | 共通 | terrain 境界から内側へ戻る。 |
| `WanderDesire` | 共通 | 明確な目標がないときに波形・乱数で徘徊する。 |
| `ManaFieldAttractionDesire` | 共通 | mana field が高い方向へ寄る。 |
| `DamageAvoidanceDesire` | 共通 | 被ダメージ後に危険方向から離れる。 |
| `RestDesire` | 将来 | 夜間休眠など。現時点では未実装候補。 |
| `ReproductionDesire` | 将来 | 繁殖期や追加スポーン判断。現時点では未実装候補。 |

## Steering

Desire の移動方向を補正する。

| 機能 | 対象 | 役割 |
| --- | --- | --- |
| `ProportionalNavigationSteering` | 捕食 | 獲物速度を見て比例航法ベースで追尾方向を補正する。 |
| `TerrainVectorProjector` | 共通 | 移動ベクトルを地形面へ投影する。 |

## Action

mana、health、damage、field、threat map などへ副作用を出す。

| 機能 | 対象 | 役割 |
| --- | --- | --- |
| `GrassEatAction` | 草食 | 草に接触したとき mana を吸収する。 |
| `CorpseEatAction` | 捕食 | 死体から mana を吸収する。 |
| `FieldManaAbsorbAction` | 共通 | mana field から定期的に mana を吸収する。 |
| `ChargeAttackAction` | 捕食 | 突進攻撃を行う。 |
| `BiteAttackAction` | 捕食 | 噛みつき攻撃を行う。 |
| `MeleeAttackAction` | 捕食 | 近接攻撃を行う。 |
| `CounterAttackAction` | 将来 | 被ダメージ時の反撃。 |
| `RandomEvasionAction` | 将来 | 乱数ベースの回避。 |
| `ThreatPulseEmitter` | 捕食/共通 | 存在または攻撃時に脅威度マップへパルスを出す。攻撃コンポーネント導入時に自動導入する。 |
| `PredatorPhaseEvolutionAction` | 捕食 | mana field に応じて predator -> highpredator -> dominant へ遷移する。 |
| `CreatureDeathState` | 共通 | 死亡、分解、死体化、死亡時 field 放出を扱う。 |

## Magic Action / Evasion

魔法関連は、通常攻撃と回避系とは別カテゴリとして扱う。

| 機能 | 対象 | 役割 |
| --- | --- | --- |
| `MagicAttackAction` | 魔法適性個体 | mana を消費して魔法攻撃を発動する。 |
| `MagicProjectileAttackAction` | 魔法適性個体 | 属性弾を生成し、対象へ遠距離攻撃する。 |
| `MagicImpactAction` | 魔法/共通 | 着弾時の damage、mana field、heat field、wind field 反映を行う。 |
| `MagicEvasionAction` | 魔法適性個体 | 空間転移、風回避など魔法による回避行動を行う。 |
| `MagicDefenseAction` | 魔法適性個体 | 被弾軽減、属性耐性、障壁などを行う。 |
| `MagicCooldownState` | 魔法適性個体 | 魔法発動間隔、発動可否、mana コスト判定を管理する。 |

## Motor

実際の移動と姿勢制御を担当する。

| 機能 | 対象 | 役割 |
| --- | --- | --- |
| `GroundMotor` | 共通 | 地形に沿って移動し、速度、旋回、慣性を制御する。 |
| `CreatureMotorBootstrap` | 共通 | Rigidbody、Collider、初期姿勢制御を整える。 |
| `MovementTelemetry` | 共通 | 移動要求、加速、減速、旋回負荷を記録する。 |

## Genome / Level 対象

| 機能 | enabled | level の意味 |
| --- | --- | --- |
| `FoodVisionSense` | 餌を視覚で見つけるか | 視野距離、視野角、探索波形の強さ。 |
| `PredatorVisionSense` | 捕食者を検知するか | 検知距離、記憶時間。 |
| `PreyVisionSense` | 獲物を視覚で見つけるか | 検知距離、追跡維持距離。 |
| `FoodDesire` | 餌へ向かうか | 餌ベクトルの重み。 |
| `PreyChaseDesire` | 獲物を追うか | 追跡ベクトルの重み。 |
| `ThreatAvoidanceDesire` | 脅威から逃げるか | 恐怖重み、逃走閾値。 |
| `BoundaryAvoidanceDesire` | 境界回避するか | 境界反発の強さ。 |
| `WanderDesire` | 徘徊するか | 好奇心、徘徊波形の強さ。 |
| `ManaFieldAttractionDesire` | mana field に寄るか | mana 勾配への誘引強度。 |
| `ChargeAttackAction` | 突進攻撃するか | ダメージ係数、コスト、判定範囲。 |
| `BiteAttackAction` | 噛みつき攻撃するか | ダメージ、コスト、判定範囲。 |
| `MeleeAttackAction` | 近接攻撃するか | ダメージ、コスト、判定範囲。 |
| `FieldManaAbsorbAction` | field 吸収するか | 吸収速度、半径、変換率。 |
| `MagicAttackAction` | 魔法攻撃するか | 発動条件、コスト、射程、属性適性。 |
| `MagicEvasionAction` | 魔法回避するか | 発動条件、コスト、回避距離、成功率。 |

`ThreatPulseEmitter` は gene の直接対象から外し、攻撃コンポーネント導入時に自動導入する。

`ZigzagEvasionSteering` は廃止予定のため、設計後の搭載対象から外す。

## 自動導入される依存機能

| 導入元 | 自動導入 | 理由 |
| --- | --- | --- |
| `ChargeAttackAction` | `ThreatPulseEmitter` | 攻撃発生時の脅威度マップ反映に必要。 |
| `BiteAttackAction` | `ThreatPulseEmitter` | 攻撃発生時の脅威度マップ反映に必要。 |
| `MeleeAttackAction` | `ThreatPulseEmitter` | 攻撃発生時の脅威度マップ反映に必要。 |
| `MagicAttackAction` | `MagicCooldownState` | 魔法発動可否と mana コスト判定に必要。 |
| `MagicProjectileAttackAction` | `MagicCooldownState` | 遠距離魔法の発動間隔管理に必要。 |

## 後回しにする機能

- ツリー状の配列による一斉 AddComponent 制御。
- GUI から component set を編集する機能。
- 繁殖期、睡眠、夜行性。
- 遠距離攻撃。
- 群れ意図、共有ターゲット、リーダー。
- 学習結果を個体内で保持して次世代へ渡す仕組み。
