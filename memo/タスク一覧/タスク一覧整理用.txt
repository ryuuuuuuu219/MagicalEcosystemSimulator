タスク一覧整理用

目的:
既存のタスク一覧を、実装本線と「やりたいこと」の別軸に分けて整理する。
今回の前提は、群知性や多対多戦闘より先に魔法関連システムを設計すること。
実ディレクトリの本線順は 1 → 2 → 3 → 4 → 5 に並べ替え済み。

---
基本方針

1. 本線タスクは依存順に置く。
- 1.知覚・地形・記憶
- 2.個体戦闘・IFF・状態遷移
- 3.魔素基盤
- 4.属性魔法
- 5.群れ意思決定・集団戦術

2. 「やりたいこと」は本線から独立した別軸とする。
- 命名は `X-n.名称`
- 横断テーマ、将来案、演出案、観測案を格納する
- 本線の依存順を壊さない

3. 群知性は戦闘だけでなく魔法とも接続する前提で後段に置く。
- 多対一 / 多対多の判断は、個体戦闘と魔法手段の両方が揃ってから設計する

---
本線タスクの整理

1.生態系コア強化
- 縄張り / 拠点ベース巡回の追加
- 座標好感度（場所記憶）評価の導入
- 高所・見晴らし評価の行動重み化
- threat map / heat field の行動意思決定統合
- 森と視界隠蔽の導入

補足:
- 「視覚隠蔽」は独立項目ではなく知覚基盤の一部として扱う
- ここでは「見える / 隠れる / 記憶する / 場を選ぶ」を扱い、戦闘フェーズ自体は持ち込まない

2.戦闘システム拡張
- 肉食の待ち伏せ条件追加
- 追跡放棄条件の精緻化
- 肉食動物→肉食動物戦闘の正式化
- 摂食対象の制御をホワイトリスト式からブラックリスト式に変更
- 種族ID・派閥ID・IFF実装
- 高脅威判定の導入
- 撤退条件の導入
- 個体戦闘の状態遷移整理

補足:
- ここでは「誰と戦うか」「いつ交戦するか」「いつ止めるか」を固める
- 後段の群知性は、この個体戦闘ルールを共有・拡張する層として扱う

3.魔素基盤構築
- 魔素グリッドの定義
- 拡散・吸収・消費の更新式実装
- 濃度差による能力補正実装
- 生体行動と魔素場の相互作用定義
- UI観測項目の最小露出

補足:
- 群れ戦術より先に、環境場としての魔素を定義する
- 将来の群知性では「どこで戦うか」「どこへ誘導するか」の判断に魔素場を使えるようにする

4.属性魔法拡張
- 属性体系（火 / 氷 / 雷 / 風 / 空間）の効果定義
- 属性発動条件とクールダウン設計
- 戦闘 / 移動 / 環境との相互作用実装
- 観測UIで属性発動の差分可視化
- 学習・進化系への接続ポイント整理

補足:
- 群知性より前に「個体が使える手段」を増やす
- その後に、群れが魔法をどう役割分担するかを 5 で扱う

5.群知性導入
- GroupIntent データ構造定義
- 指揮者個体（リーダー）選出ロジック
- 追従 / 包囲 / 離脱判断の共有規則
- 群れ内部の意図共有プロトコル
- 戦闘フェーズとの連動
- 魔法手段を含む多対一戦闘の段階判断
- 魔法手段を含む多対多戦闘の判断基準整理

補足:
- 包囲・追い込み・囮は群れ戦術の主タスクとしてここへ集約する
- 多対多戦闘は、魔法の有無で前提が変わるため 3 と 4 の後に置く

---
X 系フォルダー

X-1.やりたいこと_戦術演出
- 包囲の見え方を強くしたい
- 追い込みの演出を入れたい
- 囮が分かるようにしたい
- 森への誘導や待ち伏せを印象的にしたい
- 魔法連携が分かる見た目にしたい

X-2.やりたいこと_判断ログ
- 判断基準のログ形式定義
- 状態列挙の見直し
- ログから追跡開始 / 継続 / 放棄 / 撤退 / 包囲 / 魔法発動が読めるようにしたい

X-3.やりたいこと_陣形
- 陣形の定義
- 多対多の陣形判断基準
- 側面展開、密集、散開の見え方整理
- 魔法使用個体を含む隊形案

X-4.やりたいこと_遺伝子拡張候補
- 陣形判断基準のゲノム追加
- 群れ判断に関する遺伝子候補
- 魔法適性と役割分担の遺伝子候補
- 将来的な遺伝システムへの統合メモ

X-5.やりたいこと_将来拡張
- 支配種段階で必要になる集団知性
- 森・魔法・資源支配をまたぐ挙動案
- 現段階では土台未整備のため保留する案

---
整理時の注意

- 本線は「できる順」を優先する
- 群知性は「最後にまとめる」のではなく、「魔法込みの最終的な集団判断層」として後段へ置く
- X 系は発想メモ置き場として使い、本線の粒度を壊さない
- 5 の README は、戦闘だけでなく魔法運用との接続を書く前提にする

---
暫定結論

- 本線順は 1 → 2 → 3 → 4 → 5 に変更した
- 5 を先に詰めるのではなく、魔法関連の基盤と手段を先に設計する
- その上で群知性を「非魔法の集団戦」ではなく「魔法も含めた集団意思決定」に拡張する
