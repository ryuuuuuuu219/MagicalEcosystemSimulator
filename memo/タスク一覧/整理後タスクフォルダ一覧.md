## 方針

- active task は「今後実装・接続・検証が必要な単位」で切る。
- 完了済み、保護資料、将来案は active task から分ける。
- 旧 behaviour 参照は `Assets/script/Ingame/Creatures/before`、organ 参照は `Assets/script/Ingame/Creatures/after/organ` に読み替える。
- 1フォルダにつき、`README.md` に「目的 / 現状 / 残タスク / 対象スクリプト / 完了条件 / 移植元」を置く。
