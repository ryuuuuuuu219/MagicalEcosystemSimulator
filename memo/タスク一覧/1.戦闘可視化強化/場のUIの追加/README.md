# 場のUIの追加

## 目的

- 場の値、勾配、方向、表示 ON/OFF を UI から確認・切替できるようにする。
- 戦闘観測カメラと重ねて、指定個体周辺の場の状態を見られるようにする。

## 追加候補

- 魔力場表示 ON/OFF。
- 属性場表示 ON/OFF。
- heat / threat / mana の表示切替。
- 指定個体周辺の場の値表示。
- 勾配方向やベクトル場の矢印表示。
- tensor field の簡易デバッグ表示。

## 置き場

- 個体周辺の値は State View。
- 全体表示の ON/OFF は Properties または専用 UI ボタン。
- 表示専用のデバッグ切替は UIボタン追加側と接続する。
