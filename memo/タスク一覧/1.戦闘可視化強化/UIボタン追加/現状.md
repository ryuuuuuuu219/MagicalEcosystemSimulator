# 現状

## メインメニュー

- オブジェクト一覧、Generation、Properties を開閉できる。
- 主要ブランチを切り替えると、他のブランチや子パネルを閉じる。

## オブジェクト一覧 / State View

- 草食、肉食、資源などを一覧から選択できる。
- 選択中オブジェクトの基本状態を表示できる。
- 草食は HP、エネルギー、死亡状態、DNA コードなどを確認できる。
- 捕食は HP、エネルギー、死亡状態などを確認できる。
- 状態表示ページを切り替えられる。

## Generation

- 一斉世代更新を実行できる。
- Genome Viewer で保存済みゲノムを確認できる。
- Genome Injector で DNA 入力からスポーンできる。

## Properties

- 実行中の全体設定をスクロール式パネルで調整できる。
- Environment:
  - Time Scale
  - Terrain Size
  - Water Height
  - Initial Herbivore / Predator
  - Gen Grass Count
  - Plant Radius
- UI:
  - Selectable Layer
  - Gauge Status
  - Gauge Species
  - Gauge Alpha
- Camera:
  - Focus Mode
  - Center Top
  - Random Focus
- Generation:
  - Evaluation Axis
  - Input Modes
  - Target Phase
  - Crossover
  - Mutation
  - Mutation Chance
  - Parent Range Scale
  - Save Output
  - Run
- Deferred:
  - Perf Overlay
  - Drop Log
  - Drop Log Target
