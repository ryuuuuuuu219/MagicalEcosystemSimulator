実装予定マップまとめ

現状確認日: 2026-03-11

このメモは、構想メモではなく「実装済み / 部分実装 / 未実装」を切り分けるための中間整理用。

【実装済み】

・地形生成
・植物相
・草食動物
・肉食動物
・草食動物への攻撃
・草食動物の防御反応
・炭素リソース循環
・死亡と分解
・世代交代 UI
・DNA ビューア
・DNA 文字列からの草食個体スポーン
・オブジェクト追跡 UI
・状態表示 UI
・パフォーマンス監視スクリプト

【部分実装】

・巡回
  明示的な巡回ルートや縄張りは未実装だが、wander 波形による徘徊はある。

・肉食動物の戦闘アルゴリズム
  草食への追跡、接近、攻撃、死体摂食、クールダウンはある。
  対肉食、包囲、撤退判断、群れ戦闘は未実装。

・脅威度マップ
  `threatmap_calc.cs` と `ThreatMapsGenerator.cs` は存在する。
  ただし、現行 AI の主行動ループに全面統合されている状態ではない。

・世代交代
  評価軸、交差、突然変異、入力元切替、保存済み genome 反映はある。
  UI 上の導線整理と複数親の高度な評価は未整理。

【補助マップ案】

脅威度マップ（threatmap）
・空間 / エリアの危険度を取得する
・上位捕食者が周囲の危険度を上昇させる
・低脅威方向への退避候補に使う
・現状は単独スクリプトとして存在し、今後 AI 本体へ統合する

座標の好感度マップ
・高度
・見晴らし
・遮蔽率
・群れとの距離
・草原 / 森林といった地形差
を座標スコアに変換して使う想定
